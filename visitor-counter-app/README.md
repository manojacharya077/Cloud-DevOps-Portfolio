# Multi-Service Web Deployment
## Docker Compose | Python | Redis

A containerized full-stack architecture demonstrating microservices orchestration.

### Key Features
* **Orchestration:** Used Docker Compose to manage a multi-container environment (Python/Redis).
* **Networking:** Isolated container networks for secure internal communication.
* **Configuration:** Proficient use of port mapping and environment variables for local/production parity.

### Tech Stack
* **Languages:** Python (Flask)
* **Database:** Redis
* **Orchestration:** Docker Compose
